我仅仅使用了UIKit来完成此Playground. 同时，本Playground使用了数学计算公式来实现动画。

本项目是通过代码来模拟一款名叫万花尺的儿童玩具。

I only use UIKit to complete this playground. At the same time, this Playground uses mathematical calculation formula to achieve animation.

This project uses code to simulate a children’s toy called a Spirograph. The Spirograph is a drawing toy, composed of two parts: an external drawing board and an internal circular drawing board. The inner circle board is like a gear, with many pen holes at different radius along the center of the circle. The outer board is a large circular hole similar to the inner gear. The inner circular board is placed in the circular hole of the outer drawing board and rotates along the circumference. A regular pattern like a flower can be drawn from the pen hole with a pencil.

This project uses mathematical formulas to calculate the position of a point at a specific time, and draws the coordinates on the view.
